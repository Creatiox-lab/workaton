Librería Ubidots MQTT para ESP32

Introducción

Esta es la documentación para la librería Ubidots MQTT para ESP32 en español para la plataforma Arduino, modificada desde su repositorio original, con la finalidad de crear compatibilidad con las placas basadas en ESP32. También se ha optimizado para que el sistema no sea bloqueante (en el código original hay bucles infinitos si no hay conexión a WiFi).

Dependencias

Esta libreria depende de PubSubClient, por favor asegúrese de instalarla, o descargarla y añadirla a su proyecto.

Ejemplos

Para revisar ejemplos de esta librería, dirigirse a la carpeta "ejemplos" dentro del mismo directorio. Estos han sido creados utilizando Visual Studio Code, con la extensión PlatformIO IDE.
